# m-claude-plugins

Collection of Claude Code plugins implementing a self-learning agent system.

Extends Claude Code into a personal AI manager:
- **Context** — Dynamic context in `~/.claude/context/`
- **Supervisor** — Highest-level agent that coordinates and delegates
- **Determinism** — Playbooks (commands, skills) over improvisation
- **Self-learning** — Captures learnings, evolves system
- **Long-term memory** — Integration with PKM (Obsidian)
